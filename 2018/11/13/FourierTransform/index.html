Region of convergence ( ROC )

The region that satisfy follow criteria, we call it region of convergence.
$$\sum_{k=-\infty }^{+\infty }\left | x(k)z^{-k} \right |<\infty$$
different z-transform have different ROC, so when we want to determine the z-transform, we have to point out the ROC.

Properties

  1. The ROC is a ring or a disk centered around the origin. It cannot contain any poles and must be a connected region.
  2. If $x(k)$ is a right-sided sequence, the ROC extends outward from the outermost pole to $|z| < ∞$.
  3. If $x(k)$ is a left-sided sequence, the ROC extends from the innermost pole to $z=0$.
  4. The FT of $x(k)$ converges absolutely if and only if the ROC includes the unit circle.

Linear shift invariant system ( 1D )

System map one or more input signals onto one or more output signals. In the case of a signal input and s signal output we have.

z-transform analysis of LSI system

$$Y(z)=H(z)X(z)$$

$$H(z)=\frac{B(z)}{\tilde{A}(z)}=\frac{\sum_{v=0}^{N}b_vz^{-v}}{\sum_{\mu =0}^{M}\tilde{a}_ \mu z^{-\mu }}=\frac{\sum_ {v=0}^{N}b_vz^{-v}}{1-\sum_{\mu =1}^{M}a_\mu z^{-\mu }}=b_0\frac{\prod_{v=1}^{N}(1-z_{0,v}z^{-1})}{\prod_{\mu =1}^{M}(1-z_{\infty ,\mu}z^{-1})}$$
we have set $\tilde{a}_ 0=1$ and $\tilde{a}_ k=-a_k$. we distinguish the following case:

  1. general recursive ( zeros and poles )
    $$H(z)=\frac{B(z)}{1-A(z)}=\frac{\sum_{v=0}^{N}b_vz^{-v}}{1-\sum_{\mu =1}^{M}a_\mu z^{-\mu }}=\frac{z^M}{z^N}\frac{\sum_{v=0}^{N}b_vz^{N-v}}{z^M-\sum_{\mu =1}^{M-1}a_\mu z^{M-\mu }}$$
  2. non-recursive ( all zeros )
    $$H(z)=B(z)=\sum_{v=0}^{N}b_vz^{-v}=\frac{\sum_{v=0}^{N}b_vz^{N-v}}{z^N}$$
  3. purely recursive ( all poles )
    $$H(z)=\frac{b_0}{1-A(z)}=\frac{b_0}{1-\sum_{\mu =1}^{M}a_\mu z^{-\mu }}=\frac{b_0 z^M}{z^M-\sum_{\mu =0}^{M-1}a_\mu z^{M-\mu }}$$
    Note that the frequently used terminology in the brackets does not account for zeros and poles at $z = 0$.
  4. H(z) does not uniquely specify the impulse response of the system. We must also specify an ROC.
  5. If the system is stable, the impulse response must be absolutely summable. Therefore, the ROC must include the unit circle.
  6. If the system is causal, the ROC must extend from the outermost pole out- ward.
  7. For a stable and causal system, we must require that all poles lie within the unit circle.
  8. For a stable and invertible system, we must require that all poles and all zeros are within the unit circle. Systems which meet this condition are also called minimum-phase systems.
