java笔记

Posted on 七月 27, 2020
  1. do while循环先执行循环体，再进行判断
  2. continue之后直接停止当前，执行下一个循环，而break是直接直接结束整个循环
  1. 数组的变量名arr相当于一个指向堆内存地址的指针，直接输出数组变量名的话就会输出地址，new是为该数组分配内存，第二行的arr2作为一个地址，将arr的内存地址赋予arr2的话，这样子，arr2指向的内存地址和arr相同，实际上这两成为了公用内存的数组
  2. 源码(.java文件)-—JDK——>字节码（.class文件）——JVM——>机器码
  3. java和c++，还有c的对比应该是要区分的

  4. java的泛型

  5. 当创建 String 类型的对象时，虚拟机会在常量池中查找有没有已经存在的值和要创建的值相同的对象，如果有就把它赋给当前引用。如果没有就在常量池中重新创建一个 String 对象。
