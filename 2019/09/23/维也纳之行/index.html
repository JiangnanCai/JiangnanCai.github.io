【🇦🇹】维也纳之行

Posted on 九月 23, 2019 本文总阅读量次

几个月前，听闻菜头要来欧洲开会，心想着如果考试顺利结束的话，可以去找他一波。如同预料的一般，考试一波三折，但是还是顺利的结束了，目前应该是没有挂科的，如果按照剧情发展，下学期应该是能够写毕设的。

所以9月16号一考完试，便马上和菜头联系好了在维也纳的碰头，买上了大巴车票，订好了青旅，就等着出发的时候了。

第一天

18号晚上从科隆坐大巴到维也纳，真亏我想得出来，晚上11点半从科隆机场出发，第二天早上10点半才到维也纳。当然，主要还是因为资金的问题，不得不忍受了。坐到第二天早上，感觉屁股都已经不是自己的了。到维也纳刚好中午的时间，因为菜头19号的晚上7点多才到，所以我19号下午大概就是所谓的自由活动时间了，直接去青旅check in也太无聊了，所以想着自己先在维也纳逛一下，但是又为了避免和明天的行程发生冲突，所以不应该去那些热门景点，但是我其实对维也纳有什么景点都还不是很清楚。

就直接坐地铁到了维也纳市中心，也就是斯蒂芬大教堂那个站，出了地铁口，就能看到斯蒂芬大教堂了，应该算是维也纳标志性的教堂了，进去瞧了瞧，貌似是在做弥撒，也懒得拿出相机了，毕竟之后肯定还要来，就直接离开了，准备找间金拱门坐一下，对付了下午餐，然后搜索了一下维也纳的景点，路上遇到了黑死病纪念柱（图二）。

最终选择了菜头大概应该不回去的Albertina博物馆，里面有莫奈，毕加索和丢勒的馆藏。

在这个美术馆我新认识了一个画家「Maria Lassnig」玛丽亚·拉斯尼格，是一个以自画像和身体意识的理论而闻名的奥地利画家。上面的前两幅作品就是她的，第一幅是「女性拉奥孔」，就是之前在梵蒂冈看到的那个非常出名的雕像的再创作。第二幅是「自画像和雨伞」。第三幅图就是小朋友在老师的带领下鉴赏毕加索画作的场景，第四幅就是睡莲达人莫奈的作品（话说莫奈画的睡莲也太多了吧！）。

比较遗憾的是，因为今天有什么派对的缘故，丢勒的那几幅名作都只用于私人派对展出，没有公开展览，只有明天才能看到。得知这个消息，感觉非常沮丧，虽然学生票才10欧，但是还是非常亏的感觉。

离开albertina博物馆，我就直接去青旅check in了，整个下午呆在青旅玩王者，晚上去维也纳主火车站把菜头和他同学接了回来。

第二天

第二天的行程都是菜头定的，第一站是美泉宫，到了美泉宫，发现巨多人，想要在自动售票机上买票，发现需要等2小时才能够进去，于是考虑了一下，在美泉宫外面拍了几张照片，果断放弃了美泉宫。

第二站是霍夫堡宫，好像也没有进去，在宫殿外面转悠了几圈，本来因为霍夫堡宫对面的小教堂会有跳蚤市场，想要进去看看，结果发现时间不对，要过个几天才开始。

所以最后准备离开了，反而路过了奥地利国家图书馆，8欧的门票，便进去了，一个古典的藏书室，emmmmm，怎么说呢，感觉不太值。不过顶部还是挺好看了，也有很多比较古老的藏书和地图，只是我不太了解这些东西，老藏书室的后面就是现代的奥地利国家图书馆了。

第三站是维也纳艺术历史博物馆，这个感觉是最值的，买门票的老爷爷也非常好心，没怎么看我们的学生卡就给了我们学生价。里面有很多名家的作品，例如维米尔，丢勒，彼得勃鲁盖尔之类的。
上面的图一二都是勃鲁盖尔的名作，第一幅是「雪中猎人」，第二幅是「巴别塔」，他的画作主要艺术题材是农村的生活，给我的感觉是他画中的人物都非常的小。我也是在这里第一次认识这位画家，虽然好像美术书中就有巴别塔的图片。

至于图三的维米尔，我之前就比较熟悉了，这幅作品的名字叫做「画室」，之前我就听说过他的作品「戴珍珠耳环的少女」和「挤奶女工」。图四是一位大叔在临摹画作（有丶厉害！）。

中午去吃了个越南的米粉Pho，还行，小米椒下多了，汤底有点辣。

下午就去了上美景宫，因为资金问题，就没有去下美景宫。瞻仰克里姆特的名作「吻」，当然除了他的作品，还有很多名家的作品，例如他关门弟子席勒的一些作品，来到了维也纳，我才第一次听多了维也纳分离派这个名词。

「吻」前面站的人是做多的，一般来说，一家美术馆只有要一幅名作，就能够吸引来大量的人流，我相信大多数去卢浮宫的人都只是为了「蒙娜丽莎」而去的。图二就是克里姆特的弟子席勒的画作「拥抱」。

逛完上美景宫之后，我们就开始去吃晚餐了，我推荐的是德国的猪肘子，24块钱，两人份的，结果三个人都没吃完，真的分量非常大和油腻，不过吃得一本满足

吃完晚饭往金色大厅那边去，结果在门口被票贩子给骗了，其实我早该想到，那个家伙是骗子，只是没能够确定，一开始一直告诉我们现在有优惠，自己是官方员工，买两个人的票就能送多一个人，而且学生身份能够升一个位置等级，我们估计是被这种蝇头小利冲昏了头脑，我问他是在最大那个厅举办吗，他左右而言他，没有直接回答我的问题，而是说可以提前来进行拍照（真是坑爹，后来才知道就一个50平米左右的小房间，有什么好拍的），等我们刷完卡付完钱之后，才拿出地图，说他们的舞台并不在金色大厅，而是在几条马路之后的小街里面。回过神来真的非常坑，相当于21欧一个人，但是提供的表演真的和街头表演差不多，想起学语言的时候在克虏伯别墅是看的10欧的音乐会，排场和观众和大厅都要比这个高上好几个档次。今晚的经历真的不堪回首🙈。

看完昏昏欲睡的表演出来之后，路过金色大厅外，有公屏在免费直播金色大厅内的歌剧表演，还有椅子提供，除了在户外可能会有点冷，简直完爆之前的表演。有非常多的人坐在屏幕下欣赏歌剧。不得不说，维也纳的娱乐生活真的非常丰富。

第三天

今天菜头的同学回美国了，只剩下我和菜头了，按照原计划，早上6点多我们就赶到维也纳主火车站了，因为今天的计划是去布达佩斯逛一天，虽然看起来不是很远，但是维也纳离布达佩斯也是要坐3个小时大巴的，感觉和珠海到阳江花的时间差不多，中午10点半左右到的布达佩斯，因为实在是肚子太饿了，就在车站买了点面包吃，因为匈牙利不用欧元，我们也没有换福林，所以就只能直接刷卡了。

中午我们往市中心里去，准备先吃个午饭，去市中心总归是能够找到吃的。想着布达佩斯最著名的国会大厦一定是在市中心，所以就直接坐地铁往那边去了。

多瑙河畔还是非常棒的，只是现在正值正午时分，日头正烈，菜头提议我们在河边的马路上走上一段，享受一下河岸的风光，只是这日头，让我难以忍受，在正面绕了国会大厦一圈之后，我们就准备去找吃的了，因为吃不惯西餐的缘故，我们近些天吃的基本都是亚洲的菜色，今天中午就找了一家拉面店。躲在街头巷角的某个地下，门口非常不起眼，貌似是个日本老爷爷开的，怎么说呢，味道一般吧，点了个酱油拉面，有点咸的感觉。

接着就是坐车上山，去渔人堡。

山上人是不少，还有一个教堂，只是这个教堂进去居然要收费，我来欧洲这么久，去了这么多个教堂，除了巴塞罗那的圣家堂，这还是第二个，简直难以置信。本来想要登上教堂的钟楼去俯瞰对面的多瑙河岸的，但是因为下一波得要3点才能够进去，所以时间应该是没办法让我们泡上温泉的，于是放弃。本来还想在渔人堡一层的咖啡厅喝咖啡欣赏风景的，但是这里的咖啡厅只能够使用现金，而且不收欧元，于是也就此作罢。
