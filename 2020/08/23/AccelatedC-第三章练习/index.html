一开始一头雾水，不知道怎么结束对vector类型homework的输入，后来上网才发现，的确只要输入一个非double类型的变量，例如char类型，就可以使得流出现错误状态，从而结束循环。

3-1

假设我们希望找出一个数值集合的中值；同时假定到目前为止，我们已经读进一些数值了，而且不清楚还要读进多少个值。
证明：我们不能丢掉已经读到的任何值。丢掉任意值都没办法找到中值（文字论述题）。
提示：一个可行的证明策略是，先假定我们可以丢掉一个值，然后找出我们的集合中未读的（也就是未知的）那部分数值，要求这些数值将会使中值恰好就是我们丢掉的那个值。

3-2

把一个整数集合分成个数相等的四部分，而且第一部分含有的整数值比其他各部分的都大，第二部分的值比第一部分的小，比其他两部分的大，剩下的两部分则是以此类推。按照上面的要求，编写一个程序来计算并且打印这四部分。

int main() {

vector<string> word;
vector<int> count;
string x;

typedef vector<int>::size_type vec_size;
vec_size word_size = word.size();

cout << "please enter some words: " << endl;

while (cin >> x){
    for(vec_size i = 0; i < word_size; i++) {
        if (x == word[i]) {
            count[i]++;
        } else {
            word.push_back(x);
            count.push_back(1);
        }
    }
}

if(word_size == 0){
    cout << "no word!";
    return 1;
}

for(vec_size i = 0; i < word_size; i++){
    cout << "word [" << word[i] << "] appear "
        << count[i] << " times!";
}
return 0;

}
